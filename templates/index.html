About the Model

What is HomeLens AI?

A machine learning regression model trained on real estate data to estimate property prices based on location, size, amenities, and market indicators.

Exploratory Data Analysis — generated from your actual training data

Built to solve a real problem

Buying or selling a house without accurate price data leads to poor decisions. HomeLens AI uses an XGBoost regression model trained on {{ meta.train_n + meta.test_n }} real estate records to deliver instant, data-driven valuations.

  • Location-aware predictions

    City and locality are deeply encoded into the model as key features.

  • Size & layout intelligence

    Area, BHK, bathrooms, and floor level all drive the prediction.

  • Hyperparameter-tuned

    Model was optimized using cross-validation and grid search for best generalization.

  • Fully reproducible

    Every step — data, training, evaluation — lives in one Jupyter notebook.

Why Trust Us

Reasons to trust HomeLens AI

We built this with transparency, validation, and real-world data — not guesswork.

Real market data

Trained on {{ meta.train_n + meta.test_n }} actual property records across 6 major Indian cities — no synthetic toy data.

Rigorously tested

5-fold cross-validation plus a completely separate test set. Model never saw test data during training.

Transparent predictions

Every estimate comes with a confidence range. R²={{ meta.r2 }}, MAPE={{ meta.mape }}% — all shown, nothing hidden.

Tuned for India

Feature encoding handles Indian city names, localities, and area units (sq ft / BHK) natively.

Open methodology

Full notebook, training pipeline, and model weights are available on GitHub for anyone to inspect.

Outperforms baselines

XGBoost was chosen after comparing 5+ algorithms — it scored highest on every metric that matters.
